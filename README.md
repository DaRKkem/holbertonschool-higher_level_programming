# Higher-Level Programming -- Python, SQL, and JavaScript

![Python](https://img.shields.io/badge/Python-3.12-blue) ![MySQL](https://img.shields.io/badge/MySQL-8.0-orange) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow) ![License](https://img.shields.io/badge/License-Holberton-yellow)

This repository contains the Higher-Level Programming curriculum at Holberton School, covering Python, SQL, and JavaScript. The curriculum spans from fundamental programming concepts through advanced object-oriented design, database management, data serialization, REST API development, and front-end DOM manipulation.

## Sub-Projects

### Python (15 projects)

| Project | Description |
|---------|-------------|
| python-hello_world | Basic printing, string formatting, slicing |
| python-if_else_loops_functions | Control flow, loops, functions |
| python-import_modules | Module imports, sys.argv |
| python-data_structures | Lists, tuples, list operations |
| python-exceptions | Try/except, raising exceptions |
| python-classes | OOP, classes, attributes, encapsulation |
| python-more_classes | Advanced Rectangle class, __str__, __repr__, __del__ |
| python-inheritance | BaseGeometry, Rectangle, Square inheritance |
| python-input_output | File I/O, JSON serialization, CSV |
| python-more_data_structures | Set operations, dictionary manipulation, lambda |
| python-test_driven_development | Doctests, unit tests, TDD |
| python-everything_is_object | Python object model quiz |
| python-serialization | JSON, Pickle, CSV, XML serialization |
| python-abc | Abstract Base Classes, duck typing, mixins |
| python-server_side_rendering | Jinja2 templating, file/DB data sources |
| python-object_relational_mapping | MySQLdb driver + SQLAlchemy ORM |
| restful-api | REST API design, Flask, HTTP methods, security |

### SQL (2 projects)

### JavaScript (2 projects)
