============================
Test for 5-text_indentation.py
============================

This module defines a function ``text_indentation(text)``.

Importing the function:
    >>> text_indentation = __import__('5-text-indentation').text_indentation


Basic usage
===========

    >>> text_indentation("Hello world. How are you?")
    Hello world.
 
    How are you?

Multiple punctuation
====================

    >>> text_indentation("First line: second line. Third line?")
    First line:
 
    second line.
 
    Third line?

Leading and trailing spaces
===========================

    >>> text_indentation("  Space before.  Space after? ")
    Space before:
 
    Space after?

Empty string
============

    >>> text_indentation("")

TypeError if not string
=======================

    >>> text_indentation(123)
    Traceback (most recent call last):
    ...
    TypeError: text must be a string

    >>> text_indentation(None)
    Traceback (most recent call last):
    ...
    TypeError: text must be a string

Multiple lines, multiple punctuations
=====================================

    >>> text_indentation("Lorem ipsum. Quonam modo? Non autem hoc: Plus semper voluptatis?")
    Lorem ipsum.
 
    Quonam modo?
 
    Non autem hoc:
 
    Plus semper voluptatis?
